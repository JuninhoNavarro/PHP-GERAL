
# Calculadora de Semestres e Tabuada Interativa
## Sobre o Projeto

Este projeto foi desenvolvido como exercício prático de PHP e lógica de programação, com o objetivo de aplicar conceitos de formulários, cálculos, e estruturas de repetição (for).

O sistema solicita ao usuário:

Seu nome

O semestre atual em que está cursando

Um número de 1 a 10

Após o envio, os dados são processados por um script PHP que:

Calcula quantos semestres faltam para concluir o curso (considerando um total de 8 semestres).

Gera a tabuada do número escolhido usando um laço for.

Exibe as informações e a tabuada dentro de um card estilizado em HTML.

## Tecnologias Utilizadas

HTML5 – Estrutura e formulário

CSS3 – Estilização dos cards e layout

PHP – Processamento de dados e geração da tabuada com for

## Funcionalidades

Formulário com campos de nome, semestre atual e número para tabuada

Cálculo automático de semestres restantes

Geração dinâmica de tabuada de 1 a 10 com for

Exibição do resultado em cards estilizados

Após o envio, as informações são tratadas por um arquivo PHP, que realiza:

O cálculo de quantos semestres faltam para a conclusão do curso

E a exibição dos dados em um card estilizado em HTML

Este trabalho foi realizado com o objetivo de repor uma atividade perdida devido à ausência e, ao mesmo tempo, serviu como revisão dos principais conteúdos que serão cobrados na prova, contribuindo para o reforço do aprendizado.
